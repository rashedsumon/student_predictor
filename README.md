# Student Performance in Mathematics Predictor AI

An interactive end-to-end Machine Learning web application deployed using Streamlit Cloud. It fetches performance datasets on-the-fly and runs a trained Random Forest Regression pipeline model to predict standardized mathematics outcomes.
